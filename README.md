# DayZ Server Files

A curated collection of **DayZ Expansion Market** configuration files
designed for server owners who want a balanced, easy-to-maintain
economy.

## Features

-   Expansion Market compatible
-   Balanced General Trader and Black Market pricing
-   Support for vanilla and popular modded content
-   Consistent pricing structure
-   Organized by category for easy management

## Installation

1.  Copy the JSON files into your Expansion Market profile.
2.  Assign them to the correct traders.
3.  Restart your server.
